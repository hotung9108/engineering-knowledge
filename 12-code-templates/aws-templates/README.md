# AWS Code Templates & Infrastructure as Code (IaC)

> Ready-to-use boilerplate code, Terraform scripts, and AWS CDK templates for real-world production setups.

---

## Content

Thư mục này dùng để lưu trữ các đoạn code, scripts, hoặc template có thể tái sử dụng nhanh chóng cho các dự án thực tế (theo đúng rule của `12-code-templates`).

Các ví dụ có thể lưu ở đây:
- `terraform-vpc-base/`
- `cdk-serverless-api/`
- `boto3-python-scripts/`
- `github-actions-aws-deploy.yml`

*(Mỗi template phải có README.md hướng dẫn cách dùng, requirements và cách setup credentials).*
