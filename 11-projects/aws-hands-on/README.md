# AWS Hands-on Projects

> Practical integration projects to practice deploying, configuring, and architecting solutions on AWS.

---

## Content

Thư mục này dành cho việc thực hành (hands-on) kết hợp nhiều dịch vụ AWS lại với nhau thành các dự án thực tế (theo đúng rule của `11-projects`).

Bạn có thể tạo các dự án mẫu tại đây, ví dụ:
- **Serverless Web App**: Kết hợp S3 (Frontend) + API Gateway + Lambda + DynamoDB.
- **VPC & EC2 Auto Scaling**: Setup mạng VPC, ALB, và Auto Scaling Group cho EC2.
- **CI/CD Pipeline**: Sử dụng CodePipeline, CodeBuild, CodeDeploy.

*(Mỗi project nên có file README.md hướng dẫn step-by-step cách triển khai và sơ đồ kiến trúc).*
