# 01 Data Fundamentals

> The foundational concepts of Data Engineering, focusing on how data is structured, modeled, and organized for analytics at scale.

## Overview

Before building complex data pipelines, it's crucial to understand how to model data so that it can be queried efficiently by analysts and business intelligence tools. This section covers the fundamental building blocks of data warehousing and modern dimensional design.

## Core Concepts

- **OLTP vs OLAP**: The difference between transactional databases (optimizing for fast writes) and analytical databases (optimizing for fast, complex reads).
- **Dimensional Modeling**: How to structure data into Facts and Dimensions to make it understandable and fast to query.
- **Data Vault**: An advanced modeling technique designed for enterprise agility and historical tracking.
