# 02 AWS Deep Dive

> The comprehensive guide to mastering Amazon Web Services (AWS) for Production environments.

## Overview

AWS is the dominant cloud provider in the world. To be a Cloud Engineer or DevOps Engineer, you must deeply understand its core building blocks: Identity, Networking, Compute, Storage, and Databases. This section breaks down the most heavily used AWS services, stripping away the marketing jargon to focus on how they actually work under the hood.

## Core Categories

- **Security & Identity**: IAM (The gatekeeper to everything).
- **Networking**: VPC, Route53, CloudFront.
- **Compute**: EC2, ECS (Containers), EKS (Kubernetes), Lambda (Serverless).
- **Storage**: S3, EBS, EFS.
- **Databases**: RDS (SQL), DynamoDB (NoSQL), ElastiCache (In-memory).
- **Analytics & Big Data**: Redshift, EMR, Athena.
- **Messaging & Event-Driven**: SQS, SNS, EventBridge, API Gateway.
