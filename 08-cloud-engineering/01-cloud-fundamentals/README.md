# 01 Cloud Fundamentals

> The foundational concepts of Cloud Computing, focusing on deployment models, shared responsibility, and the transition from on-premise to the cloud.

## Overview

Before diving into specific AWS or GCP services, it is critical to understand what "The Cloud" actually is. Cloud computing is the on-demand delivery of IT resources over the internet with pay-as-you-go pricing. This section covers the fundamental models that dictate how much you manage versus how much the cloud provider manages.

## Core Concepts

- **Deployment Models**: Public Cloud, Private Cloud, and Hybrid Cloud architectures.
- **Service Models**: IaaS (Infrastructure as a Service), PaaS (Platform as a Service), and SaaS (Software as a Service).
- **Shared Responsibility Model**: Understanding exactly where AWS's security responsibility ends and your security responsibility begins.
